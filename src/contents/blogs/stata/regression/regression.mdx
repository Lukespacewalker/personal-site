# Logistic regression
‡∏ó‡∏ö‡∏ó‡∏ß‡∏ô‡∏Ñ‡∏ì‡∏¥‡∏ï‡∏®‡∏≤‡∏™‡∏ï‡∏£‡πå‡∏û‡∏∑‡πâ‡∏ô‡∏ê‡∏≤‡∏ô‡∏Ç‡∏≠‡∏á Logarithm ‡∏Å‡∏±‡∏ô‡∏Å‡πà‡∏≠‡∏ô ‡πÇ‡∏î‡∏¢‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô Log ‡∏ô‡∏±‡πâ‡∏ô‡∏à‡∏∞‡∏°‡∏µ Domain ‡∏Ñ‡∏∑‡∏≠ $(0,\infty)$ Range ‡∏Ñ‡∏∑‡∏≠ $(-\infty,\infty)$

log_b(1) = 0 \\
log_b(0) = undefine
$$
$$
log_b(x) + log_b(y) = log_b(xy)  \\
log_b(x) - log_b(y) = log_b(\frac{x}{y})

‡πÇ‡∏î‡∏¢ $P$ ‡∏Ñ‡∏∑‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏°‡∏ô‡πà‡∏≤‡∏à‡∏∞‡πÄ‡∏õ‡πá‡∏ô (Probability) ‡∏ó‡∏µ‡∏à‡∏∞‡πÄ‡∏Å‡∏¥‡∏î‡πÄ‡∏´‡∏ï‡∏∏‡∏Å‡∏≤‡∏£‡∏ì‡πå‡∏ô‡∏±‡πâ‡∏ô ‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô‡∏à‡∏∞‡πÄ‡∏´‡πá‡∏ô‡∏ß‡πà‡∏≤ $\frac{P}{1-P}$ ‡∏Å‡πá‡∏Ñ‡∏∑‡∏≠‡πÅ‡∏ï‡πâ‡∏°‡∏ï‡πà‡∏≠ (Odds) ‡∏Ç‡∏≠‡∏á‡πÄ‡∏´‡∏ï‡∏∏‡∏Å‡∏≤‡∏£‡∏ì‡πå‡∏ó‡∏µ‡πà‡πÄ‡∏£‡∏≤‡∏™‡∏ô‡πÉ‡∏à ‡πÅ‡∏•‡∏∞‡πÄ‡∏£‡∏≤‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÄ‡∏Ç‡∏µ‡∏¢‡∏ô‡πÉ‡∏ô‡∏£‡∏π‡∏õ‡∏Ç‡∏≠‡∏á Logit ‡πÑ‡∏î‡πâ‡∏î‡πâ‡∏ß‡∏¢
