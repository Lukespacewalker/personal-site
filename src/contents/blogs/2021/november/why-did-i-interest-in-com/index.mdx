# ตอนที่ 1 

ได้จับคอมพิวเตอร์ตอนแรกน่าจะ อนุบาล 1-2 คือทีทำงาน พ่อแม่ คือ สสจ. คอมจอขาวดำ มีปุ่ม TURBO (ซึ่งถ้ากดแล้วจริงๆ คอมจะช้าลง) ไปเล่น เกมกอล์ฟและต่อยมวย ที่ทำงานตอนเย็นๆ (ตอนนั้นยังมีพิมพ์ดีด ผมก็ชอบไปกดให้ก้านพิมพ์มันเข้ามาติดกัน สนุกดี)
พ่อซื้อคอม CPU 80486 เลยได้เล่นคอมคือเล่นเกม ตอน อนุบาล 3 และเล่น Word กับ Powerpoint ตอน ป.1-2 คือเอา Clipart มาแปะๆ เป็นรูป และทำ slides อะไรมั่วๆไป
แม่ซื้อหนังสือ HTML มาให้ลองทำตอนนั้นน่าจะ ป.1-2 ยังใช้ Windows 3.11 อยู่ ไม่มี Browser เลยเปิดไฟล์ HTML ไม่ได้
พ่อซื้อคอม CPU Intel Pentium 3 500Mhz และ GPU NVIDIA RIVA TNT2 Model 64 ตอน ป.3 ตอนนั้นเป็น Windows 98-Me ก็ลองหัดเขียนเว็บโดยใช้ Microsoft FrontPage ซึ่งตอนนี้ไม่มีแล้ว และ Macromedia Dreamweaver (ตอนนี้เป็นของ Adobe ไปแล้ว) ตอนนั้นมันเป็นแบบ Drag drop ยังไม่เข้าใจ HTML เท่าไหร่ (ตอนนั้นมัน HTML3 มั้ง) แล้วอยากทำลุกเล่นเลยไปซื้อหนังสือ Javascript มาอ่านและก็ทำตาม (ถ้าหาหนังสือเจอจะถ่ายรูปให้ดู) จากนั้นเริ่มรู้ว่ามันต้องมีเบื้องหลัง ที่มันเรียกว่า CGI เลยไปซื้อหนังสือ ASP:Active Server Page 3.0 มาอ่าน แล้วลองทำตาม ปรากฎว่า Windows Me มันดันไม่มี IIS:Internet Information Service (ก็คือไม่มี Server) เลยรันไม่ได้  ก็พับไป
